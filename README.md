This project is a simple Python-based Guessing Game developed as part of CodeCraft Infotech Internship.

The program generates a random number within a given range and asks the user to guess it. After each guess, the program provides feedback indicating whether the guess is too high or too low. The game continues until the user correctly guesses the number.

Additionally, the program tracks and displays the total number of attempts taken by the user to guess the correct number.

This project demonstrates the use of core Python concepts such as:
- Random module
- Loops (while)
- Conditional statements (if-elif-else)
- User input handling
- Variables and operators
